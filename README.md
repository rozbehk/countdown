Count It Down
==========

The game
--------

This program receives the information of the movies and series available on the TMDB website with the API and after storing it in the database shows the date of the release of the movie or the airing of the series as a countdown.
admin have a search panel for searching in TMDB database and add movies to the website

TECHNOLOGIES
--------------
* HTML
* Nodejs
* css
* Mongo
*javascript
